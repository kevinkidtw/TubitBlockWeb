# tubitblock-tools

The firmware for boards.

## Version

    v1.5.0

## Firmware list

- Arduino

    1. Arduino UNO - configurable firmata
    2. Arduino UNO Ultra - configurable firmata (for those customized board which has A6 A7 pins)
    3. Arduino Mega 2560 - configurable firmata

- Micro Python

    1. Esp32 - standard micro python v1.25.0
    2. Esp32S3 - standard micro python v1.25.0
    3. Esp8266 - standard micro python v1.25.0
    4. K210 - maixpy_v0.6.3_2_gd8901fd22_openmv_kmodel_v4_with_ide_support
